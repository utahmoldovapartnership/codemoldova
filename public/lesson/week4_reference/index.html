About this reference

You do not need to memorize every tag. Goal: when you open an HTML file (yours, AI-generated, or a template), you can read what each part does.

Why semantic tags?

Tags like header, nav, and main describe meaning, not just appearance. Prefer them over bare div when the role is clear.

  • Landmarks help accessibility tools navigate your page.
  • Clear structure makes AI edits more reliable.
  • CSS can target semantic tags consistently.
