# Moved: use the GitHub portfolio template

The in-class portfolio is now the **Next.js** template repo:

```bash
git clone https://github.com/utahmoldovapartnership/porfolio-template.git yourname-portfolio
cd yourname-portfolio
npm install
npm run dev
```

See the [README on GitHub](https://github.com/utahmoldovapartnership/porfolio-template#readme) for Vercel deploy and `data/projects.ts` fields.

The old plain HTML starter files were removed from this folder.
